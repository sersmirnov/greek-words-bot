# 🇬🇷 Greek Words Bot

Telegram-бот, который каждый день в 10:00 по кипрскому времени присылает в группу 10 случайных греческих слов уровня A1-A2 с переводом, транскрипцией и примером.

## Деплой на Railway — пошагово

Всё делается через браузер, без терминала. Время — **~15 минут**.

### Шаг 1. Создать бота в Telegram (2 мин)

1. В Telegram открыть [@BotFather](https://t.me/BotFather)
2. Отправить `/newbot`
3. Имя: `Greek Words Daily` (или любое)
4. Username: должен заканчиваться на `bot`, например `larnaca_greek_bot`
5. **Скопировать токен** — выглядит как `1234567890:ABCdef...`
6. Отправить BotFather `/setprivacy` → выбрать бота → **Disable** ← важно, иначе бот не увидит команды в группе

### Шаг 2. Залить код на GitHub (3 мин)

1. Зайти на [github.com](https://github.com), создать новый репозиторий (можно Private)
2. Имя: `greek-words-bot`
3. На странице репозитория нажать **"uploading an existing file"** (или **Add file → Upload files**)
4. Перетащить туда **все файлы** из папки `greek_bot`:
   - `bot.py`
   - `words.json`
   - `requirements.txt`
   - `railway.json`
   - `runtime.txt`
   - `.gitignore`
   - `README.md`
5. Внизу нажать **Commit changes**

⚠️ **НЕ загружай** файл `.env` если случайно создал его — там твой токен!

### Шаг 3. Деплой на Railway (5 мин)

1. Зайти на [railway.com](https://railway.com), залогиниться через GitHub
2. При регистрации потребуется привязать карту — это норма, спишут только если выйдешь за $5 кредитов первого месяца
3. На дашборде нажать **+ New Project → Deploy from GitHub repo**
4. Авторизовать Railway → выбрать репозиторий `greek-words-bot`
5. Railway начнёт билд автоматически. **Подожди 1-2 минуты**, чтобы первый деплой завершился
6. Перейти в **Variables** (слева) и добавить переменные окружения:

| Variable | Value |
|---|---|
| `BOT_TOKEN` | твой токен от BotFather |
| `TIMEZONE` | `Asia/Nicosia` |
| `SEND_HOUR` | `10` |
| `SEND_MINUTE` | `0` |
| `WORDS_PER_DAY` | `10` |

`CHAT_ID` пока **не добавлять** — получим на следующем шаге.

После добавления переменных Railway автоматически передеплоит сервис.

### Шаг 4. Получить CHAT_ID группы (2 мин)

1. Добавить бота в свою группу с друзьями
2. В группе сделать бота **админом** (или дать право Send Messages)
3. В группе написать: `/chatid@твой_бот_username`
4. Бот ответит ID — что-то вроде `-1001234567890`
5. Если бот не отвечает — проверь, что в Шаге 1 ты сделал `/setprivacy → Disable`

### Шаг 5. Добавить CHAT_ID и финал (1 мин)

1. Вернуться в Railway → твой проект → **Variables**
2. Добавить переменную `CHAT_ID` со значением из шага 4 (например `-1001234567890`)
3. Railway передеплоит автоматически
4. В группе написать `/words@твой_бот_username` — бот должен прислать 10 слов
5. Если работает — на следующий день в 10:00 утра придёт автоматически 🎉

## Проверка и логи

В Railway: твой проект → **Deployments** → клик на последний деплой → **View Logs**

Если в логах видишь `Бот запускается...` и `Планировщик запущен` — всё ок.

## Команды бота

- `/start` — приветствие
- `/words` — получить 10 слов сейчас
- `/chatid` — показать ID текущего чата
- `/about` — информация о боте

## Стоимость

- **Первые 30 дней:** бесплатно (Railway даёт $5 кредитов)
- **После:** ~$3-5/мес (бот ест мизер ресурсов, но Hobby plan имеет минимальный платёж $5/мес)

## Изменить расписание / количество слов

В Railway → Variables → поменять `SEND_HOUR`, `SEND_MINUTE` или `WORDS_PER_DAY` → Railway передеплоит сам.

## Добавить свои слова

1. На GitHub открыть `words.json`
2. Нажать карандашик (Edit)
3. Добавить новые объекты по тому же формату
4. Commit changes
5. Railway увидит push и передеплоит за минуту

## Локальный запуск (для теста)

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# вписать BOT_TOKEN
python bot.py
```
